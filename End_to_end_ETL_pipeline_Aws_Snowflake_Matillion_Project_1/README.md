# End-to-End Data Pipeline using Matillion, AWS S3 & Snowflake
Designed and implemented an end-to-end data pipeline using Matillion ETL, integrating data from AWS S3 (Excel files) into Snowflake for analytics and reporting.
The pipeline involves extracting Excel data from S3, transforming it using Matillion components such as Query, Table Iterator, and KPI Calculator, and dynamically managing workflows through variables. Processed data is then loaded into Snowflake tables, enabling efficient storage and scalability.
Finally, the curated dataset is connected to Power BI for building interactive dashboards and visualizing key business metrics.
